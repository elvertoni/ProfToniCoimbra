# Aula 1 - Introducao ao HTML

HTML e a linguagem base usada para estruturar paginas web. Nesta aula, vamos entender como os elementos se organizam, como o navegador interpreta as tags e por que uma boa estrutura ajuda no desenvolvimento front-end. O foco e dar repertorio pratico para a turma iniciar a construcao de paginas simples com clareza.

## O que e HTML

<div class="callout c-blue">
 <div class="callout-icon">🧠</div>
 <div class="callout-body">
 <p class="callout-title">Estrutura</p>
 <p class="callout-text">HTML organiza o conteudo da pagina em elementos semanticamente identificaveis.</p>
 </div>
</div>

HTML descreve titulos, paragrafos, listas, links, imagens e outras partes da pagina. Ele nao cuida do estilo visual nem da logica de comportamento; sua funcao principal e definir a estrutura do conteudo.

## Como o navegador le uma pagina

<div class="callout c-amber">
 <div class="callout-icon">⚠️</div>
 <div class="callout-body">
 <p class="callout-title">Interpretacao</p>
 <p class="callout-text">O navegador interpreta as tags para montar a arvore da pagina.</p>
 </div>
</div>

Quando o navegador encontra um arquivo HTML, ele identifica abertura e fechamento de tags, organiza o documento internamente e renderiza o conteudo para o usuario. Uma estrutura mal formada pode causar erros de exibicao ou dificultar manutencao.

## Exemplo pratico

Um arquivo com `h1`, `p` e `a` ja permite construir uma pagina simples de apresentacao. Esse primeiro passo ajuda o estudante a perceber que toda interface web comeca por uma base estrutural bem definida.

 <div class="callout-icon">💡</div>
 <div class="callout-body">
 <p class="callout-title">Analogia</p>
 <p class="callout-text">HTML funciona como a planta de uma casa: define onde cada parte deve ficar.</p>
 </div>
</div>
